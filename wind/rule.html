遊戲規則

一.遊戲設置

  1. 選角色

    2. 每位玩家分發二張角色牌，每位玩家挑選其中一張作為自己的角色，當所有玩家都挑選好角色後，潛伏的角色牌要面朝下蓋伏，其他角色面朝上亮出。剩餘的角色牌都面朝下堆疊在一旁。

  2. 抽陣營

    3. 在挑選角色之後，拿取等同於玩家人數的身份牌（參照下表）隨機分給每位玩家一張身份牌，所有玩家的身份牌都是保密的（面朝下放置）。

    人數 潛伏 軍情 醬油
    3 1 1 1
    4 1 1 2
    4 2 2 0
    5 2 2 1
    6 2 2 2
    6 3 3 0
    7 2 2 3
    7 3 3 1
    8 3 3 2
    9 3 3 3

  3. 起始手牌

    4. 將遊戲牌洗勻後，分給每位玩家三張牌為起始手牌。之後將剩餘的遊戲牌放在中央，作為牌庫。

二.開始遊戲

  1. 回合開始階段

    2. 通常可以直接跳過，除非有角色技能在此階段使用。

  2. 抽牌階段

    3. 從牌庫頂抽二張牌。當牌庫的牌抽完時，馬上把棄牌堆洗混後形成新的牌庫。

  3. 出牌階段

    4. 可以使用任意張牌，每回合必須且只能傳出一張情報

  4. 情報傳遞階段
    1. 傳遞中

      2. 情報在兩個玩家之間，此時可使用【調虎離山】【截獲】【鎖定】。

    2. 情報抵達

      3. 情報在玩家面前，此時可使用【破譯】【轉移】；不可使用【調虎離山】【截獲】【鎖定】。

    3. 情報接收

      4. 決定是否接收。此時不能使用任何卡牌和人物技能。

  5. 出牌階段

    6. 同上，但不可再傳遞情報

  6. 回合結束階段

    7. 通常可以直接跳過，除非有角色技能在此階段使用。

三.結束遊戲

    只要有一方陣營達成勝利條件，遊戲立即結束，該陣營獲勝

    潛伏：一位隊友搜集到3張紅色情報。

    軍情：一位隊友搜集到3張藍色情報。

    醬油：完成自己角色卡上的機密任務。

    *醬油方各自為不同陣營

四.死亡 & 輸掉遊戲

  1. 當玩家無法在自己回合中傳出情報，就會輸掉遊戲，輸掉遊戲不會發動角色死亡時觸發的效果。

  2. 若自己回合再傳出情報前已滿足死亡或勝利條件時，則視為死亡或勝利並不視為輸掉遊戲。

  3. 當獲得三張或以上的假情報時，視為死亡。某些角色死亡時會觸發效果。

  4. 在同時滿足死亡及勝利的條件下，死亡優先於勝利，某些角色技能除外
